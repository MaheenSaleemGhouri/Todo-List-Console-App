# Quickstart: Todo Viewing Application

## Setup

1. Ensure Python 3.13+ is installed on your system
2. Navigate to the project directory
3. Run the application using: `python src/main.py`

## Usage

To view all todos, run the application and use the appropriate command as implemented. The application will display all todos with their ID, title, description, and completion status in a clear, readable format.

## Example Output

The console will display todos in a tabular format similar to:

```
ID          Title           Description           Status
1           Sample Task     This is a sample...   Pending
2           Another Task    More details here     Completed
```

## Requirements

- Python 3.13+
- No additional dependencies required
- Console/terminal access
